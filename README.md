CRC：粗(Cū)算 人(Rén)工 成(Chéng)本，以「上海社平工资」为长远发展的锚定物，即： 1 CRC = 上海社平工资 <br/>
抹零：CRC抹零抹到百位（例如：12399.00 -> 12300.00）， 由CRC计算的价格也统一抹零抹到百位， 免去争议

## 云端规则

### 云号

「云号」一般应是认证机构缩写，初次签约时和我方确定的，一经确定终身不能变改，极端情况的变改走特殊程序 <br/>
每个「云号」需要缴纳一个「云号授权费」，每年按规定缴纳**一个**「年费」维系使用，否则视为放弃使用注销云号

#### 云号后缀

分区后缀  |代表类别
---------:|----------
-A        |体系认证
-B        |产品认证
-C        |服务认证

其他情况其他后缀，具体情况具体协商

#### 云号域名

每个云号需要绑定一个域名，该域名由认证机构持有、以及保持续费，需要按我方要求解析，通过该域名使用系统

#### 云号示例

假定一个机构的缩写是：AQA <br/>
那么体系认证的云号是：AQA-A

如果再做产品认证，其云号是：AQA-B <br/>
如果再做服务认证，其云号是：AQA-C

AQA-A、AQA-B、AQA-C虽然同属于一家机构，但视作不同使用单元，需各自分别缴纳各自的授权费、年费等费用

#### 云号授权费

按照猫科动物命名，划分授权等级，不同授权等级对应不同的「云号内有效证书数上限」、以及不同的「标准年费」

### 大模型-符文

拥有「云号」之后，还需要购买相应的「大模型」，依托「大模型」建立起应用（项目分类），才能进一步的使用 <br/>

## 导入实现方式

#### 程序脚本方式导入

我方依据认证机构掌握的、提供的数据结构文件，执行程序脚本，完成数据导入

#### 多人填表方式导入

我方向认证机构提供一整套的Excel，认证机构按约定的多人协同规则，完成填表

#### 旧系统的哈希算法

如果能够清楚原旧系统密码的哈希算法，那就可以良好实现新旧过度，可以使用原账号原密码登入新系统 <br/>
如果不能清楚，那只能统一重设新密码
